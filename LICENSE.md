# 🍔 Sistema de Cantina

Sistema web desenvolvido para gerenciamento de uma cantina escolar, permitindo cadastro de usuários, autenticação, carrinho de compras, realização de pedidos e gerenciamento administrativo.

## 👨‍💻 Autor

**Matheus Cruz**

Projeto desenvolvido utilizando:

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

---

## 📋 Funcionalidades

### Usuário

- Cadastro de usuários
- Login e logout
- Visualização de produtos
- Adição de produtos ao carrinho
- Remoção de produtos do carrinho
- Finalização de pedidos
- Tela de pagamento

### Administrador

- Gerenciamento de produtos
- Gerenciamento de usuários
- Controle de pedidos
- Visualização das vendas

## ⚙️ Instalação

### 1. Clone o projeto

```bash
git clone https://github.com/Math-Cruz12/projectwebvascocoutinho.git
```

### 2. Copie para o XAMPP

Mover a pasta para:

```text
xampp/htdocs/
```

### 3. Criar banco de dados

Abra o phpMyAdmin e crie um banco chamado:

```sql
cantina
```

### 4. Importar banco

Importe o arquivo:

```text
database/cantina.sql
```

### 5. Executar

Abra o navegador:

```text
http://localhost/projectwebvascocoutinho
```
